회사 소개

VisualCamp는 혁신적인 시선 추적(Eye Tracking) 기술을 개발하는 선도 기업입니다. 다양한 산업에서 활용되는 시선 데이터 분석 솔루션을 제공하며, 사용자 경험 향상과 데이터 기반 의사결정을 돕고 있습니다.

나의 역할

저는 VisualCamp에서 알고리즘과 여러가지 프로그래밍을 공부하며
내년에 회사에 업무관련으로 도움이 되기 위해 공부하는 막내입니다.

주요 하는 일

  • VisualCamp에서 회의 참석: 현재 회사에서 지금 현재 필요점과 보안점 관련 회의 참석
  • 알고리즘 공부: 프로젝트 오일러로 문제를 풀고 이 코드가 어떻게 작동되는지 설명
  • 자료구조 공부: 효율적으로 문제 해결과 코드 성능을 향상을 위해서 따로 공부한다

성과 및 배운 점

- 문제를 풀고 답인 코드를 설명을 하며 이 코드에 대해 어떻게 실행되는지 이해할수 있다
- 실제 직장 생활 환경과 사회를 알게 되었다
